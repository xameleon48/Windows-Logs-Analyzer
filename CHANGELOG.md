# Changelog

Все значимые изменения в этом проекте будут отражены в этом файле.

## [1.0.0] - 2024-03-27

### 🎉 Первый стабильный релиз

#### ✨ Добавлено
- **Поиск** по тексту, IP-адресам, именам процессов
- **Временные фильтры**: StartTime, EndTime, LastHours, LastDays, Today, Yesterday
- **Фильтрация** по EventID, уровню событий (Error, Warning, Information, Critical), имени журнала
- **Параллельная обработка** (UseParallel, MaxThreads) для ускорения анализа
- **Очистка пустых файлов** (CleanEmptyLogs, DryRun, MinFileSizeKB)
- **Экспорт** в CSV и HTML с поддержкой кириллицы (UTF-8 BOM)
- **Статистика**: топ EventID, распределение по уровням
- **Подсветка** найденного текста в консоли и HTML
- **Таймауты** на проблемные файлы (TimeoutSeconds)
- **Цветной вывод** в консоли
- **Прогресс-бар** для последовательной обработки
- **Подробная справка** (Help)

#### 🐛 Исправлено
- Корректная обработка кириллицы в CSV и HTML
- Обработка ошибок при чтении поврежденных файлов
- Пропуск пустых файлов для ускорения
- Таймауты для зависших файлов

#### 📝 Документация
- Полное README с примерами
- Справка в скрипте (-Help)
- CHANGELOG.md
- install.ps1 для легкой установки

#### ⚡ Производительность
- Оптимизация XPath фильтров
- Параллельная обработка до 16 потоков
- Фильтрация на уровне чтения для временных диапазонов
